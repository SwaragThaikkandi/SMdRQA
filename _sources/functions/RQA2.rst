RQA2
====

The ``RQA2`` module provides a modern, object-oriented interface for
Recurrence Quantification Analysis (RQA).  It supersedes the procedural
functions in :mod:`SMdRQA.RQA_functions` and adds surrogate-data testing and
chaotic-system simulation in a single coherent package.
